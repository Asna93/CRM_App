# CRM Application

A full-stack Customer Relationship Management (CRM) application developed as a collaborative team project.

## Overview

The CRM system helps businesses manage leads, companies, contacts, tasks, and customer interactions through a centralized dashboard.

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* Django
* Django REST Framework
* JWT Authentication

### Database

* PostgreSQL

## Features

* User Registration and Login
* Authentication and Authorization
* Company Management
* Lead Management
* Deal Management
* Ticket Management
* Notifications
* Dashboard and Reporting

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Environment Variables

Create a `.env` file and configure:

```env
SECRET_KEY=
DATABASE_URL=
JWT_SECRET=
```
